Christmas Card

This interactive christmascard was made together with Arthur Godthelp in an assignment for creative technology M2 in the course Programming & Physical

    Instructions for the interactive christmascard:
  • click on the canvas to make the snowglobe "shake".
  • hover over the fireplace with the mouse, a candle should appear.
  • Press the "L" key when then candle is on the log, the brown ellipse.

    • The tool used to create this card was Processing version 4.3, revision 01293. To make it work on a website the processing language is coverted to JavaScript, which makes it possible to have the interactiveness on a website. The link down below is to download the orginial processing file, as a zip file. To open this file you need to have processing installed on your device.
